# GXDE 25.3 更新日志（2025.11.08）

![GXDE 25.3](/news/25.3/desktop.jpg)

## 更新日志

::: info
如果在更新后出现控制中心无法开启、桌面无法进入等问题，可以打开系统终端或切换至 tty 输入以下命令进行修复：
```bash
sudo apt install gxde-desktop -y
```
:::

* 修复 hetao 支线 calamares 无法正常安装系统的问题
* Kmre 支持 6.16 内核
  ![Kmre 支持 6.16 内核](/news/25.3/kmre.jpg)
* 修复 GXDE 系统助手工具箱无 GXDE 内核管理器入口的问题
* 启用 GXDE 25 mips64 构建
* 修复 mips 下默认不显示桌面壁纸的问题
* 桌面支持壁纸跨屏显示
  ![桌面支持壁纸跨屏显示](/news/25.3/wallpapers.jpg)
* 桌面支持壁纸显示天气预报（目前需手动创建文件 ~/.config/GXDE/dde-file-manager/weatherReport 以启动该功能）
* 新增 fcitx5 98 版五笔输入法
* 修复 mips64el 安装镜像在 7A1000 显卡下鼠标指针无法显示的问题
* 修复 mips64el 下新用户桌面壁纸丢失的问题
* calamares 支持 riscv64 grub-efi
* 新增 riscv64 ISO
  ![新增 riscv64 ISO](/news/25.3/riscv64.jpg)
* 修复 riscv64 hetao gxde-iso-builder 无法正常构建的问题
* 调整壁纸
* 新增中秋特别壁纸（终端输入 `sudo apt install gxde-wallpapers-mid-autumn-festival-2025` 或在星火商店安装均可）
* 修复小小电脑下部分目录重命名文件变为隐藏文件的问题（https://gitee.com/GXDE-OS/GXDE/issues/ID0SJO ）
* GXDE 桌面支持调整壁纸显示模式（https://gitee.com/GXDE-OS/GXDE/issues/ICZ2W4 ）
  ![GXDE 桌面支持调整壁纸显示模式（https://gitee.com/GXDE-OS/GXDE/issues/ICZ2W4 ）](/news/25.3/wallpapersmode.jpg)
* GXDE 启动器支持鼠标左键滑动滚动列表
* 新增初步 WSL 支持

* Kmre 支持 docker-ce
