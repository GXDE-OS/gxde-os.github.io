# GXDE 25.2 更新日志（2025.08.10）

![GXDE 25.2](/news/25.2/desktop.jpg)

## 更新日志

::: info
如果在更新后出现控制中心无法开启、桌面无法进入等问题，可以打开系统终端或切换至 tty 输入以下命令进行修复：
```bash
sudo apt install gxde-desktop -y
```
:::
