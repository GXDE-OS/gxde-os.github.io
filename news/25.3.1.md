# GXDE 25.3.1 更新日志（2026.02.12）

![GXDE 25.3.1](/news/25.3.1/desktop.jpg)

## 更新日志

::: info
如果在更新后出现控制中心无法开启、桌面无法进入等问题，可以打开系统终端或切换至 tty 输入以下命令进行修复：
```bash
sudo apt install gxde-desktop -y
```
:::

* GXDE 文件管理器支持读取部分 `autorun.inf` 缩略图
* 更新 `gl4es`、`box64`、扫描管理器、日志管理器 版本
* 修复关机、登录页面设置异常壁纸后控件与背景融为一体无法分辨的问题
* 修复 GXDE 共享文件夹工具 IP 10 网段用户生成网址异常的问题
* GXDE 硬件查看器不使用 Qt 默认标题栏，改为使用自定义标题栏，将菜单按钮移到标题栏
* GXDE 硬件查看器移除 CPU 和内存的使用率进度条
* GXDE 硬件查看器移除启动动画
* 修改 GXDE 硬件管理器配色，契合系统主题色
  ![GXDE 硬件查看器](/news/25.3.1/gxde-hardware-viewer.jpg)
* GXDE 热区新增多屏幕适配
* GXDE 控制中心多屏幕分辨率窗口设置支持显示与设置刷新率
  ![GXDE 控制中心多屏幕分辨率窗口设置支持显示与设置刷新率](/news/25.3.1/screen.jpg)
