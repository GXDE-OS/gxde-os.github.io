# GXDE 25.4 更新日志（2026.05.03）

![GXDE 25.4](/news/25.4/desktop.jpg)

经过我们几个月的不断打磨，GXDE 25.4 正式发布了  

## 更稳定的内核基线
从 25.4 开始，我们所有架构都使用 LTS 内核，均使用 6.18、6.6 的内核组合，不再跟随 Debian Backport 的内核更新，避免再出现更新内核后 dkms 炸了、虚拟机安卓模拟器显卡驱动炸了的情况  
对于飞腾 CPU 用户，使用 GXDE 也将会变得更加简单，无需再手动切换至飞腾专用的 6.6 内核，默认的 6.18 和 6.6 内核均内置了飞腾专用的驱动程序  
同时内核均修复了 CVE-2026-31431 漏洞，并集成了 NTFS PLUS 驱动程序（需自己启用）  
对于有新硬件或需要体验内核新特性的用户，如您依旧需要新版本的内核，输入命令 `sudo apt install linux-image-amd64 linux-headers-amd64` 即可安装 Debian Backport 的滚动内核，25.3.1 X86 升级上来的用户依旧是使用 Debian Backport 滚动内核  
arm 用户不推荐使用 Debian Backport 滚动内核，可能会出现问题  

## 驱动大更新
我们跟进了 Debian Sid 的部分新版本驱动，如N卡驱动（`sudo apt install nvidia-driver`）、博通驱动（`sudo apt install broadcom-sta-dkms`）  
注：建议搭配上文的 6.18/6.6 内核进行使用，使用 Debian Backport 的默认内核可能会因为内核 API 的变化导致驱动无法使用  
对于使用高通网卡的用户，我们也在 25.4 中内置了 `firmware-atheros` 固件包，修复了高通网卡无法联网的问题   
同时我们亦改善了硬件查看器的使用体验，能查看的信息更多了  
最后，系统自带的资源监视器支持查看 GPU 占用率，现在不需要第三方应用就知道我的显卡怎么样了

内置的扫描管理器、日志管理器、相机、星火应用商店、玲珑等应用都跟随上游进行了更新，体验更佳  
同时我们引入了麒麟虚拟键盘替换原先的 onboard,UI 和体验都更佳
