# GXDE 25.3.1 Changelog (2026.02.12)

![GXDE 25.3.1](/news/25.3.1/desktop.jpg)

## Changelog

::: info
If you encounter issues like Control Center failing to launch or desktop inaccessibility after updating, open the system terminal or switch to TTY and run the following command to fix:
```bash
sudo apt install gxde-desktop -y
```
:::

* GXDE File Manager supports reading thumbnails from some `autorun.inf` files
* Updated versions of `gl4es`, `box64`, Scanner Manager, and Log Manager
* Fixed issue where controls blended into the background after setting an abnormal wallpaper on shutdown/login pages
* Fixed issue where GXDE Shared Folder tool generated incorrect URLs for users on the IP 10 network segment
* GXDE Hardware Viewer no longer uses the default Qt title bar; now uses a custom title bar with menu buttons moved to the title bar
* Removed CPU and memory usage progress bars from GXDE Hardware Viewer
* Removed startup animation from GXDE Hardware Viewer
* Updated GXDE Hardware Viewer's color scheme to match the system theme
  ![GXDE Hardware Viewer](/news/25.3.1/gxde-hardware-viewer.jpg)
* Added multi-screen support for GXDE Hot Zones
* GXDE Control Center's multi-screen resolution window now supports display and setting of refresh rates
  ![Multi-screen resolution window supports refresh rate display and setting](/news/25.3.1/screen.jpg)
* Fixed scaling errors on non-wallpaper-spanning extended screens in multi-screen setups
* Updated the Windows-displayed icon for the GXDE installation image
  ![GXDE installation image icon updated](/news/25.3.1/icon.png)
* Modified dependencies for `gxde-api`, `gxde-installer-reborn`, and `dpa-ext-gnomekeyring`
* Updated Cross-Device Cooperation
* Fixed incomplete Chinese translations in `gxde-timer`
* `gxde-desktop-extra` virtual package now recommends installing `firmware-atheros`
* Updated `treeland` version
* GXDE 25 updated to kernel 6.6.119
* Fixed compilation errors of deepin-anything on kernel 6.18
* Migrated GXDE Hot Zones to Qt6
* Fixed issue where GXDE Hot Zones failed to trigger properly (https://gitee.com/GXDE-OS/GXDE/issues/ICW2NY)
* Added internationalization support to GXDE Hardware Viewer
* Fixed text overflow issue in the GXDE time plugin on the top panel
* Fixed positional error in the gxfm batch rename tool
* Adjustment: i386 machines no longer automatically refresh wallpaper cache to reduce update overhead
* GXDE Hardware Viewer now supports password-free startup
* Fixed duplicate network interface table entries in GXDE Hardware Viewer
* Fixed sound playback issue in Kmre
* Added caching functionality to GXDE Hardware Viewer and fixed English titles appearing as Chinese
* Fixed uninstall issue for Cherry Studio
* Included `firmware-atheros` firmware in ISO to fix Qualcomm wireless card connectivity issues
* Fixed blurry icons in the GXDE top panel on high-DPI screens (https://gitee.com/GXDE-OS/GXDE/issues/ICTQ7V)
* Fixed dual title bar issue in DTK6 applications
* Updated fastfetch following upstream changes and replaced fastfetch GXDE icon
  ![Updated fastfetch and GXDE icon](/news/25.3.1/fastfetch.jpg)
* Added package `deepin-ocr-plugin-manager` to the repository
* Updated Log Collector to version 6.5.25
* Updated deepin Cross-Device Cooperation to version 1.1.23
* Updated Scanner Manager to version 0.1.19
* Updated deepin Terminal to version 6.5.23
* Updated Hangover Wine to version 11.0
* Fixed blurry icons in Control Center on high-DPI screens (https://gitee.com/GXDE-OS/GXDE/issues/IBCTBH)
* Fixed blurry images in GXDE Hitokoto Screensaver on high-DPI screens
* Removed rest time display from GXDE Hitokoto Screensaver
  ![Removed rest time display from GXDE Hitokoto Screensaver](/news/25.3.1/gxde-time-screensaver.jpg)
* Fixed blurry images in GXDE Kernel Manager on high-DPI screens
* Fixed abnormal title bar display in GXDE Kernel Manager
* GXDE Taskbar Resource Monitor plugin now supports displaying CPU temperature and battery charge/discharge power (only supported on some machines, requires manual activation)
  ![Resource Monitor plugin supports CPU temperature and battery power display](/news/25.3.1/dock-plugin.jpg)
* Fixed incorrect display of battery plugin on the top panel
* Fixed issue where there were two Power options in the top panel plugin (https://gitee.com/GXDE-OS/GXDE/issues/IDH7V2)
