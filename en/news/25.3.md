# GXDE 25.3 Changelog (2025.11.08)

![GXDE 25.3](/news/25.3/desktop.jpg)

## Changelog

::: info
If you encounter issues like Control Center failing to launch or desktop inaccessibility after updating, open the system terminal or switch to TTY and run the following command to fix:
```bash
sudo apt install gxde-desktop -y
```
:::

* Fixed Hetao branch Calamares system installation failure
* Kmre now supports kernel 6.16
  ![Kmre supports kernel 6.16](/news/25.3/kmre.jpg)
* Fixed missing GXDE Kernel Manager entry in GXDE System Assistant Toolbox
* Enabled GXDE 25 mips64 builds
* Fixed missing desktop wallpapers on MIPS by default
* Desktop now supports multi-screen wallpaper spanning
  ![Multi-screen wallpaper support](/news/25.3/wallpapers.jpg)
* Desktop wallpaper now supports weather display (currently requires manually creating `~/.config/GXDE/dde-file-manager/weatherReport` to enable)
* Added fcitx5 98-Wubi input method
* Fixed missing mouse cursor on 7A1000 graphics with mips64el installation image
* Fixed missing desktop wallpapers for new users on mips64el
* Calamares now supports riscv64 grub-efi
* Added riscv64 ISO
  ![New riscv64 ISO](/news/25.3/riscv64.jpg)
* Fixed riscv64 Hetao gxde-iso-builder build failure
* Updated wallpapers
* Added Mid-Autumn Festival special wallpaper (install via `sudo apt install gxde-wallpapers-mid-autumn-festival-2025` or Spark Store)
* Fixed file renaming turning into hidden files in certain Tiny Computer directories (https://gitee.com/GXDE-OS/GXDE/issues/ID0SJO)
* GXDE desktop now supports wallpaper display mode adjustment (https://gitee.com/GXDE-OS/GXDE/issues/ICZ2W4)
  ![Wallpaper display mode adjustment](/news/25.3/wallpapersmode.jpg)
* GXDE launcher now supports left-click scrolling
* Added preliminary WSL support
  ![GXDE OS WSL installation page](/install/wsl-installer.jpg)  
  ![GXDE OS WSL fastfetch page](/install/wsl-fastfetch.jpg)  
  ![GXDE OS WSL desktop](/install/wsl-desktop.jpg)  
* Changed GXDE Daemon dependency from xbacklight to brightnessctl
* Fixed taskbar crashes under WSL
* Updated GXDE calendar holiday schedule API URL
* Kmre now supports docker-ce
* Right-click uninstaller now supports APM applications
* Added GXDE LSG support
  ![GXDE LSG demo](/gxde-lsg.jpg)  
  ![GXDE LSG Kylin demo](/gxde-lsg-kylin.jpg)  
  ![GXDE LSG Spark Store](/news/development/gxde-lsg/spark-store.jpg)
  ![GXDE LSG deepin App Store](/news/development/gxde-lsg/deepin-appstore.jpg)
* Fixed dde-grand-search compilation issue on GXDE 25 (https://gitee.com/GXDE-OS/GXDE/issues/ID2RT6)
* Updated deepin Cross-Device Cooperation to 1.1.22
  ![Updated deepin Cross-Device Cooperation to 1.1.22](/news/25.3/cooperation.jpg)
* Updated deepin-log-viewer to 6.5.22
  ![Updated deepin-log-viewer to 6.5.22](/news/25.3/logviewer.jpg)
* GXDE OCR added model file source description (https://github.com/linuxdeepin/deepin-ocr/pull/23)
* Fixed slight image movement issue in GXDE OCR initial popup (https://github.com/linuxdeepin/deepin-ocr/pull/24)
* Backup & Restore tool now supports riscv64 (https://github.com/martyr-deepin/deepin-clone/pull/45)
* Launcher now supports disabling browser sandbox
  ![Browser sandbox disable option](/news/25.3/nosandbox.jpg)
* Added new application: GXDE Hardware Manager
  ![GXDE Hardware Manager](/news/25.3/hardware-viewer.jpg)
* GXDE Calendar added 2026 holiday schedule
  ![2026 holiday schedule in calendar](/news/25.3/holiday.jpg)
* Fixed abnormal Firefox scaling issues (thanks @qaqland)
* Updated Hangover Wine to 10.18
* Updated to kernel 6.6.113
